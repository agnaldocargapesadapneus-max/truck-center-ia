# Base de Conhecimento - Grupo Truck Center Pneus (WhatsApp)

Este repositório contém a base oficial de informações do agente de vendas do WhatsApp.

## Arquivos principais
- base_conhecimento.md -> regras, identidade, fluxo e comportamento
- catalogo.json -> preços oficiais e itens do catálogo
- variaveis.json -> variáveis e parâmetros comerciais
- faq_objeções.md -> objeções e respostas padrão
- respostas_fallback.md -> respostas para casos não cobertos
- exemplos_mensagens.md -> exemplos prontos para treino
